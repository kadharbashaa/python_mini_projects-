 # Python Mini Projects

This repository contains a collection of mini Python projects ranging from beginner to intermediate level. Each project demonstrates different Python concepts, libraries, and functionality.

---

## **1. Number Guessing Game**
**File:** `numberGame.py`  
**Description:** A simple console game where the user has to guess a randomly generated number.  
**Features:**
- Random number generation
- User input validation
- Hints for high or low guesses

---

## **2. Password Manager**
**File:** `passwordManager.py`  
**Description:** A secure password manager to store and retrieve your passwords locally.  
**Features:**
- Add, view, and delete passwords
- Save passwords in a local file (`passwords.txt`)
- Simple CLI interface

---

## **3. QR Code Generator**
**File:** `QR.py`  
**Description:** Generate QR codes for text, URLs, or any string input.  
**Features:**
- Generate QR code image (`qr_code.png`)
- Supports URLs, plain text
- Easy CLI interface

---

## **4. To-Do List App**
**File:** `todo.py`  
**Description:** Simple CLI-based To-Do list application.  
**Features:**
- Add tasks
- View all tasks
- Mark tasks as done or delete

---

## **5. Crypto Price Tracker**
**File:** `Cryptotracker.py`  
**Description:** Track live cryptocurrency prices using the CoinGecko API.  
**Features:**
- Fetch current price, market cap, daily high/low, and change %
- Supports Bitcoin, Ethereum, Dogecoin, etc.
- Command-line interface

---

## **6. Rock Paper Scissors Game**
**File:** `game.py`  
**Description:** Classic Rock-Paper-Scissors game against the computer.  
**Features:**
- Random computer choice
- Score tracking
- Interactive CLI game
